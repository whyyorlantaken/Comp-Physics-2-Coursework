Basic usage: 

```sh
python orbits.py CellestialSeer --save_hdf5
```

To show the evolution:

```sh
python orbits.py CellestialSeer \
  --save_hdf5 \
  --gif_name "orbit.gif" \
  --frames 100 \
```

Full options:
